Steganography Software


There are many pieces of software that can conceal information in another file. There is a list of steganography tools by JJTC. These tools can hide pieces of information or a single file in various types of media files, such as .jpeg, .mp3. Usually, the size of the media file in which the secret is hidden will be increased somehow.

To the best of our knowledge, most of the steganography software provides both encryption and decryption functions, or will use the terms concealing and revealing. Now for some further reading to illustrate how the steganography tools work.
